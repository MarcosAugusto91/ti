1- Criando um novo componente:
Dentro de SRC temos a boa prática de criar um diretório chamado components ou componentes; Nele vamos criar componentes para que sejam importados posteriormente;
1.1- Após isso vamos criar um arquivo dentro dessa pasta com o nome do componente em PascalCase, por exemplo FirstComponent.jsx (Lembrando que a extensão atual é jsx);
Exemplo: 
const FirstComponents = () => {
    return (
        <div>
            <h1>Primeiro Componente</h1>
            <p>Este é o meu primeiro componente React!</p>
        </div>
    );
}

export default FirstComponents;

1.2 Agora basta importar em App.jsx:
import FirstComponents from './componentes/FirstComponents'
E utiliza-lo: <FirstComponents />

7- Funções de renderização:
const renderSomething = (x) => {
    if(x){
        return <h1>Renderizando isso!</h1>;
    } else {
        return <h1>Também pode renderizar isso!</h1>;
    }   
};

 {renderSomething(true)}
 {renderSomething(false)}

Nesse exemplo ao carregarmos o componente Event pelo App, logo de inicio existe a função renderSomething que é chamada no final do script passando parâmetro true ou false, que permite designar caminhos diferentes. Mas, o mais interessante é que a função é executada de fora do JSX(return) com base no parâmetro recebido dentro;
