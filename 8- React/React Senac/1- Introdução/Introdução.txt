React
*Biblioteca Javascript
*SPA (Single Page Application) -> Requisições assíncronas sem o recarregamento da página;
*A arquitetura do React é baseada em componentes; Cabeçalho, Corpo, Rodapé:
Vantagem você pode reutilizar o componente, facilita na manutenção;
*Pode ser inserido em uma aplicação ou podemos criar a aplicação apenas com React;
*É mantido pelo Meta;

Node.Js:
*É uma runtime de Javascript
*Ou seja, uma biblioteca utilizada por um compilador durante a execução do programa, oque ajuda no desempenho da nossa página;
*Construída na V8 engine (escrita em C++) da Google;
*Possibilitando criar softwares em JS no lado do servidor;
*Temos então um código JS rodando em C++ para garantir alta performance;

NPM
Gerenciador de pacotes do Node; (Php: Compose)
Vamos poder utilizar bibliotecas de terceiro;
Executar determinados scripts no nosso programa;
Dificilmente um software em Node.js não utiliza o npm;
Os módulos externos ficam numa pasta chamada node_modules;
Ela deve ser descartável, ou seja, a cada instalação do projeto baixamos todos os pacotes novamente;

1- Instalar o Node: https://nodejs.org/pt-br;
Vamos baixar um arquivo .msi, que é o instalador;
É interessante saber que o npm vem junto do Node;
Após a instalação podemos testar o Node e o npm em um terminal, para validar a instalação;
Note que os binários precisam estar no PATH, este passo está na instalação do Node.
*Baixe a versão LTS;
*Instalar;
*Abra o PowerShell para que possamos testar se a instalação funcionou:
node -v
npm -v
Caso não funcione o comando npm no powershell, você pode executar o seguinte comando e aprovar:
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
Para ver as variáveis de sistema: Pesquise no iniciar por environment e abra as variáveis de sistema;

3- NPX é um executor de scripts do node;

4- Criando uma aplicação
4.1: Comando: npm create vite@latest 
4.2: Aperte y para confirmar e em seguida coloque o nome do projeto e pressione enter
*Não utilize letras maiúsculas no nome do app
4.3: Selecione a opção React
4.4: Selecione a opção Javascript
4.5: Na opção Rolldown marque no e enter
4.6: Selecione Yes e pressione enter
4.7: Acesse a pasta do projeto criado: cd nomeDoProjeto
4.8: Instale o npm: npm install; Cria a pasta node_modules
4.9: Iniciar projetos: npm run dev

5- Estrutura do React:
node_modules: Onde ficam as dependências do projeto;
public: Assets estáticos e Html de inicialização;
src: Onde vamos programas os nossos apps;
index.html: Arquivo responsável por receber a requisição do usuário, repare que tem uma div com id:'root' onde o javascript adiciona o React ali dentro;
src/index.js: Arquivo de inicialização do React; (Hoje foi atualizado para src/main.js e ele que será inserido na div root;
src/App.js: Componente principal da aplicação; Onde vamos programar a tela inicial e demais coisas. Ela é importada em main.js que é renderizado no index.html;
